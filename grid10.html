Netflix Case Study
—
Diagram of the user and content metadata relationship for Netflix.com

The first part of the process was tracking how the user was able to interact with Netflix, then track how Netflix's API algorithm processed this information to the viewer. Examples of two different viewers experiences were provided in order to show how Netflix translated this gathered information back to the user.

This accompanied an oral presentation of my research to Paul Kahn.
